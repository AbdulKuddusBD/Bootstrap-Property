## 📖 প্রকল্প সম্পর্কে

এটি **Bootstrap 5** এর **A থেকে Z** পর্যন্ত সকল Utility Classes ও Components-এর একটি **সম্পূর্ণ বাংলা রেফারেন্স গাইড**। প্রতিটি ক্লাসের সাথে রয়েছে:

- ✅ **বাংলায় বিস্তারিত ব্যাখ্যা**
- ✅ **CSS Property ম্যাপিং**
- ✅ **Live Demo উদাহরণ**
- ✅ **Responsive Variants**
- ✅ **কোড স্নিপেট**

**🎯 কাদের জন্য?**
- ওয়েব ডেভেলপার (বিগিনার থেকে এডভান্সড)
- ফ্রিল্যান্সার
- VS Code Extension ডেভেলপার
- যারা বাংলায় Bootstrap শিখতে চান

## 📚 কভার করা বিষয়সমূহ

### 📋 A-Z বিভাগ অনুযায়ী

| বর্ণ | ক্লাস সমূহ |
|------|-----------|
| **A** | `align-*`, `alert-*`, `accordion-*`, `aspect-ratio`, `active` |
| **B** | `badge-*`, `bg-*`, `border-*`, `btn-*`, `breadcrumb-*` |
| **C** | `col-*`, `container-*`, `card-*`, `carousel-*`, `collapse` |
| **D** | `d-*` (display), `dropdown-*` |
| **F** | `flex-*`, `float-*`, `form-*`, `fs-*`, `fw-*` |
| **G** | `gap-*`, `g-*` (gutter) |
| **H** | `h-*` (height) |
| **J** | `justify-content-*` |
| **L** | `link-*`, `lh-*` (line-height) |
| **M** | `m-*` (margin), `modal-*` |
| **N** | `nav-*`, `navbar-*` |
| **O** | `opacity-*`, `order-*`, `overflow-*` |
| **P** | `p-*` (padding), `pagination-*`, `position-*`, `progress-*` |
| **R** | `rounded-*`, `row-*` |
| **S** | `shadow-*`, `spinner-*`, `sticky-*` |
| **T** | `text-*`, `table-*`, `toast-*` |
| **V** | `visible/invisible`, `visually-hidden`, `vertical-align` |
| **W** | `w-*` (width) |
| **Z** | `z-*` (z-index) |

### 🎁 বোনাস

- 📏 **Bootstrap 5 Breakpoints** (sm, md, lg, xl, xxl)
- 📐 **Spacing Scale** (0, 1, 2, 3, 4, 5)
- 🎨 **Z-Index Scale** (Dropdown to Tooltip)
